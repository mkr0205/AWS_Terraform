Create a new repository on the command line
============================================
echo "# AWS_Terraform" >> README.md

git init

git add README.md

git commit -m "first commit"

git branch -M main

git remote add origin https://github.com/mkr0205/AWS_Terraform.git

git push -u origin main

======================================================================
git remote add origin https://github.com/mkr0205/AWS_Terraform.git
